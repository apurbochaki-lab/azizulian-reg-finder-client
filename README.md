## 🚀 Key Features

* **Student Data Entry:** A simple and intuitive form to submit student names and registration numbers to the database.
* **Smart Seat Finder:** Quickly search by registration number to instantly locate exam seat allocations.
* **Neighboring Seat Display:** Automatically fetches and displays the *Previous Seat* and *Next Seat* details along with the searched seat for better classroom mapping.
* **Premium Slate Dark Theme:** A modern, highly readable, and eye-friendly dark UI built using Tailwind CSS.
* **Instant Notifications:** Real-time user feedback with beautiful success/error toasts using `react-hot-toast`, paired with localized error banners.
* **Fully Responsive Design:** Seamless user experience across all devices, including mobile, tablets, and desktops.
* **User-Friendly Info Footer:** Contains an informative guide explaining how the system works and encouraging students to contribute data.
